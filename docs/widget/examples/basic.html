Basic Widget Integration

This example demonstrates the simplest way to embed the Ask Anything widget on your page.

How it works:
  • The widget container <div id="widget-container"> defines where the widget will mount
  • The script tag loads the widget bundle
  • The data-widget-config attribute contains the configuration as JSON
  • The widget automatically initializes when the script loads

Try it out

Click the button in the bottom-right corner to open the widget and start chatting!

Integration Code

<!-- Widget Container -->
<div id="widget-container"></div>

<!-- Widget Script -->
<script 
    src="/dist/widget.js"
    data-widget-config='{
        "containerId": "widget-container",
        "apiKey": "demo-key",
        "theme": "light",
        "position": "bottom-right"
    }'>
</script>
